# gleed-landing

Next.js landing page project using a cleaned marketing template starter.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Routes

- `/` main landing page
- `/landing` same landing template route

## Project Goal

This repository is intentionally cleaned to keep the landing-page experience only.
